# Bundled data

Data files in this directory ship with the **git repository** but are
**excluded from the PyPI source distribution** (via `MANIFEST.in`). This keeps
the PyPI install lightweight while still letting `git clone` users reproduce
all paper results out of the box.

## Subdirectories

| Path | Purpose | Size | Bundled in repo? | In PyPI sdist? |
|---|---|---|---|---|
| `singapore/` | Singapore real-data tiny scenario YAML files | < 1 MB | yes | yes |
| `berlin/ortsteile/` | 96-zone Ortsteile NPZ files | ~5 MB | yes | no |
| `berlin/crosswalks/` | Block ↔ Ortsteile CSV crosswalks | small | yes | yes |
| `berlin/llm_cache_v5_4/` | V5.4 LLM cached responses | ~10 MB | yes | no |
| `shapefiles/` | Berlin block + green + water shapefiles | ~5 MB | yes | no |

## Sources + attribution

### Berlin Ortsteile NPZ files (`berlin/ortsteile/`)

Derived from the Ahlfeldt et al. (2015) "Economics of Density" replication
package via this repo's importer (`scripts/import_berlin_memory_pack.py` in
the dev repo). The NPZ files are aggregated 96-zone summaries computed from
the original block-level data; no raw third-party data is redistributed.

Citation:

> Ahlfeldt, G. M., Redding, S. J., Sturm, D. M., Wolf, N. (2015). The
> economics of density: Evidence from the Berlin Wall. *Econometrica*, 83(6),
> 2127-2189.

### Berlin shapefiles (`shapefiles/`)

`Berlin4matlab.shp` (12,309 block polygons), `BerlinGreen.shp`, `BerlinWater.shp`
are sourced from the Ahlfeldt et al. (2015) public replication package. They
are redistributed here under the original package's terms (academic / research
use). If you use these files in derivative work, please cite the Ahlfeldt et
al. (2015) paper above.

### V5.4 LLM cache (`berlin/llm_cache_v5_4/`)

Generated by this paper's V5.4 score-all-96 baseline + shock runs at seed 42
via the codex-cli LLM provider. We bundle the cache so that reviewers without
LLM credits can replay the V5.4 results in `--no-llm` mode. The cache is the
output of original work and is redistributable under the package's MIT license.
